Key Features

Game Library

Import your own game files (ROMs) via the system document picker. EmuCool automatically gives them retro-style artwork and organizes them by platform. Sort games by last played or use search to quickly find what you want.

Save Management

Each game can have up to 4 manual save slots, with optional auto-save on exit and auto-load on entry. All saves are grouped by game with thumbnail previews for easy recognition.

Game Notes

Keep separate notes for each game—like passwords, hidden secrets, or your own tips. All notes are organized by platform in the Notes tab for easy access.

Gameplay Experience

Custom virtual button layouts for each platform, placing D-pad and action buttons to feel close to original controllers. Games run full-screen with automatic pause/resume when switching apps.

Retro Visuals

Warm and retro aesthetic with a main palette of warm beige and vintage red. Background colors shift subtly when switching platforms, with game artwork overlaid with a gentle scanline effect.

Data & Privacy

All your games, saves, screenshots, and notes are stored in the app's Documents folder, accessible through the system Files app. Full control over your data with automatic file management.
